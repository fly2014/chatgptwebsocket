# chatgpt的websocket公开版   不需要翻篱笆 
>补保存token   动态聊天 
>可以公开使用   可以私密登录（name配置独立使用名字）  
>python3.6以上  

`需要安装如下东西：  安装不了假如清华园`  
`pip install openai  -i https://pypi.tuna.tsinghua.edu.cn/simple`  
`pip install flask`  
`pip install websockets`  
`pip install flask-socketio`  

>执行  
python chatgpt.py  
浏览器  
http://localhost
